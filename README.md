# Crypto Price Checker (Base Mini App)

Small Base Mini App to check live cryptocurrency prices.

This project was built to explore Base Mini Apps, Farcaster integration
and simple API communication from a frontend application.

---

## What this app does

- Allows user to enter a crypto symbol (BTC, ETH, etc.)
- Fetches current price from public API
- Displays price in USD
- Works as a Base Mini App

---

## Tech Stack

- React
- Next.js
- TypeScript
- Base Mini Apps
- Public Crypto API (CoinGecko)
